Tentang Aplikasi

Aplikasi ini mampu memberikan kesan mengenal huruf hijaiyah yang unik dan menyenangkan

Model Klasifikasi yang Digunakan

Model Convolutional Neural Network & Model Random Forest
Dalam penelitian ini, kami mengadopsi pendekatan yang berfokus pada perbandingan performa antara dua model utama, yaitu Convolutional Neural Network (CNN) dan Random Forest (RF). Dari kedua model tersebut akan dilakukan perbandingan untuk diterapkan pada aplikasi klasifikasi huruf hijaiyah.

Cara Menggunakan Aplikasi Klasifikasi

Menuliskan Huruf Hijaiyah di Canvas
Untuk melakukan klasifikasi huruf hijaiyah, pengguna perlu melakukan input dengan menggambarkan huruf hijaiyah pada area canvas yang tersedia pada halaman prediksi. Kemudian menekan tombol "Prediksi" untuk menampilkan hasil prediksi.

Akurasi Aplikasi Klasifikasi

Akurasi Tinggi diterapkan Pada Aplikasi ini
Dari hasil pelatihan model CNN dan RF terdapat perbedaan akurasi yang dihasilkan. Model CNN menghasilkan akurasi sebesar 99.7% sedangkan model RF sebesar 73.3%. Agar hasil prediksi yang ditampilkan pada aplikasi menghasilkan prediksi dengan performa yang tinggi maka model CNN diterapkan pada aplikasi klasifikasi ini.
